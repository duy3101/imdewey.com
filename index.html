Twitter Sentiment Analysis (Python)

The program asks for a word from user input. I made a parser to parse through .JSON files given from Twitter to find all the tweets that have the word in them. I used a list of sentiment values to determine a sentiment value for each tweet. By using the location that was embedded in each tweet, I calculated the average sentiment for each state in the U.S. Finally, the program generate a map of the United States that indicates a color of each state (blue to red - negative to postive).

Maze Solving Algorithm (Java)

The program takes in a 2d grid. Inside the grid, there are 0's for the walls and 1's for the paths. The letter 'S' indicates the start point and 'E' indicates the end point. I made a maze solving algorithm to find the shortest solution path as well as backtracking when there is a dead end.

Library Database (Java)

I used linked lists to create a libary database. Inside a Book class contains the ISBN number, inventory number and a back-order linked list. There is also a BookCollection class that manages the transactions (check-in, check-out) for all the books.
